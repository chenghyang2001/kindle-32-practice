① 先回顧：Cowork 的四大核心設定

就像搬進新辦公室要先把桌椅擺好、網路接上，開始委派任務之前，也要先把 Cowork 的環境設定好。

General

個人檔案與通知

設定全名、暱稱、工作類型、個人偏好（例如「用繁體中文台灣用語」），以及回應完成／Dispatch 訊息等通知開關。

Capabilities

記憶與工具載入

Memory 記憶功能、Tool access mode（工具載入方式）、Visuals（Artifacts）、Code execution 與網路存取範圍。

Connectors

連接外部服務

Web Connectors（雲端執行，如 Google Drive／Gmail）與 Desktop Connectors（本機執行，如 Control your Mac）。

③ 接回真實操作：到 Settings 實際設定一次

光做題不夠，打開 Claude Desktop → 按 Cmd + ,（或點左下角使用者名稱 → Settings）進入設定頁面，實際做一次書中的初始設定：

1️⃣ 點左側「Connectors」，找到 Google Drive 點 Configure，走一次 OAuth 授權，看狀態變成綠色的 Connected。
2️⃣ 點左側「Cowork」，找到 Global instructions 區塊，點 Edit，貼上以下內容：
所有回覆一律使用繁體中文台灣用語；刪除任何檔案之前一定要先詢問我；程式碼一律使用 Python，套件管理用 uv。
3️⃣ 點左側「Capabilities」，確認 Memory 的兩個開關（Search and reference chats／Generate memory from chat history）依你的需求開啟，Tool access mode 保持預設的 Load tools when needed 即可。

⚠️ 全域指示會套用到所有 Cowork 對話，包括排程任務和 Dispatch 派遣的任務，建議只寫通用性高的規則，不要寫太特定的內容。
