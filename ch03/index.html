① 先回顧：Cowork 的六大日常操作

New task

建立任務

一次性描述目標，Claude 自動規劃步驟、逐步執行、回報結果。輸入框下方可選專案、附檔、選模型。

Projects

專案

需要持續進行的長期工作。每個專案有自己的檔案、指示（Instructions）、排程和記憶（Memory），讓 Claude 理解完整背景。

Scheduled

排程

定期重複的工作，設定一次頻率（Daily / Weekly…），Claude 就會自動按時執行，不用每次手動觸發。

Dispatch

派遣

從手機上的 Claude app 派遣任務到桌面電腦執行，完成後結果同步回手機，適合人不在電腦前的臨時交辦。

Ideas

靈感

依「角色」（工程、行銷、財務…）分類瀏覽靈感卡片與任務範本，是快速上手 Cowork 的最佳方式。

Search

搜尋

用 ⌘K 或側欄「Search」快速搜尋過去的對話與任務紀錄，翻找累積下來的工作歷史。

展開六大功能完整比較表
功能核心用途適合情境
New task建立一次性任務臨時查詢、單次產出檔案，不需要保留工作環境
Projects管理長期持續工作需要 Claude 記住背景、檔案、偏好的專案（如旅行計畫）
Scheduled設定定期自動執行每天／每週固定重複的工作（如每日新聞摘要）
Dispatch從手機派遣任務到桌面人在外面，臨時想交辦事情給桌面電腦處理
Ideas瀏覽依角色分類的任務範本不知道能請 Claude 做什麼、想找靈感時
Search搜尋過去的對話與任務紀錄想找回之前做過的工作，忘記存在哪裡時

另外別忘了權限請求與安全機制：Claude 要使用工具或存取系統資源前，一定會先跳出對話框讓你確認，按下 Allow 才會執行，不會在你不知情的狀況下動手。

③ 接回真實操作：到 Cowork 做一次 demo

光做題不夠，打開 Claude Desktop → 切到 Cowork 分頁，把書中 3-1 節「建立第一個任務」的經典範例貼進去實際跑一次：

1️⃣ 在任務輸入區貼上：
在桌面建立一個「每日報告」資料夾，裡面放三個檔案：todo.txt（今天的待辦事項，包含 5 個項目）、meeting.md（本週會議記錄範本）、budget.csv（本月預算表，含標題列和 3 筆範例資料）
2️⃣ 確認內容無誤後按 Let's go →。Claude 會先在 Thought process 說明規劃，接著開始逐步執行，右側 Progress 面板會即時顯示 4 個步驟的進度。
3️⃣ 執行過程中若跳出「Claude wants to use create_directory from Desktop Commander」這類權限請求對話框，確認路徑安全後點 Allow，這是 Cowork 的安全機制，不會擅自動作。
4️⃣ 完成後回頭看 Claude 用繁體中文回報的結果，體會「你只描述目標、Claude 自主規劃並使用工具完成」的委派流程；再點右側 Progress 面板展開看每一步的詳細紀錄。

💡 進階挑戰：把同一個任務改成在對話框輸入 /schedule，體驗把一次性任務轉成每天自動執行的排程。
