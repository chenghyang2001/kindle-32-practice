30 個雜亂檔案 → 5 個分類子資料夾

範例 4-2 · 旅行照片整理

2,748 張照片 → 日期-國家-城市子資料夾

核心工具：Desktop Commander + Python 讀 EXIF/GPS，靠座標反查城市，比檔名判斷更準。

範例 4-3 · 家庭費用報告

28 筆收據 CSV → 3 工作表 Excel

核心工具：Desktop Commander + xlsx Skill（openpyxl 建表、LibreOffice 驗證公式）。

展開書中「4-4 本章小結」四大重點
重點內容
任務指令的撰寫技巧要具體、有條理、包含期望的輸出格式。不要只說「幫我整理」，而是說明分類規則、命名格式、要幾個工作表。指令越明確，結果越符合預期。
Connector 和 Skill 的自動選擇Claude 會根據任務需求自動載入對應工具：操作檔案系統用 Desktop Commander，建立 Excel 用 xlsx Skill，不需要使用者手動指定。
Progress 面板是任務儀表板每個任務都有清楚的步驟分解和進度顯示。完成的步驟打綠色勾勾，正在執行的步驟顯示藍色標記，隨時可知道 Claude 做到哪了。
權限機制是安全護欄Claude 第一次使用某個工具時會請求權限。「Allow for this task」是最方便的選項，同一個任務中的同類操作只需要授權一次。

③ 接回真實操作：到 Cowork 做一次 demo

光做題不夠，打開 Claude Desktop → 切到 Cowork 分頁，跑一次書中 4-1 節「整理下載資料夾」的原始任務：

1️⃣ 先準備測試環境（或直接對你真正的「下載」資料夾操作）：在桌面建一個「下載測試」資料夾，放幾十個雜亂檔案（PDF、圖片、影片、壓縮檔都行）。
2️⃣ 在任務輸入區貼上書中原始指令：
請幫我整理桌面上的「下載測試」資料夾。這個資料夾裡有 30 個雜亂的檔
案，包含 PDF、圖片、影片、音樂、壓縮檔、安裝檔、文件等各種類型。請
按照以下規則分類：
1. 依檔案類型建立子資料夾（文件、圖片、影片音樂、壓縮檔、安裝檔、其他）
2. 將每個檔案移動到對應的子資料夾
3. 完成後列出整理結果的樹狀結構
資料夾路徑：~/Desktop/下載測試
3️⃣ 按 Let's go →，觀察右側 Progress 面板展開的步驟，以及 Context 面板自動載入的 Desktop Commander Connector。第一次要求建立資料夾時會跳出權限對話框，選 Allow for this task 一次授權整個任務。
4️⃣ 完成後打開 Finder／檔案總管，對照 Claude 回報的樹狀結構，確認 30 個檔案是否都歸位到對的子資料夾，並體會它連 .CR2 這種冷門格式都能判斷「本質上是圖片」的分類能力。

⚠️ Cowork 目前是研究預覽版（research preview），第一次跑會請你確認檔案存取權限，放心它每步都會問過你。
