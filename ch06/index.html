③ 接回真實操作：到 Cowork 做一次 demo

光做題不夠，打開 Claude Desktop → 切到 Cowork 分頁，把書中範例 7 的完整指令貼進去實際跑一次：

1️⃣ 先在 Connectors 設定中確認 Web search 已啟用，然後在任務輸入區貼上：
請幫我做一份關於「2026 年台灣 AI 產業發展趨勢」的綜合研究報告。需要搜尋網路上的最新資訊，整合至少 5 個不同來源，報告內容包含：1) 產業現況概述 2) 主要技術趨勢 3) 台灣企業的佈局與投資 4) 政府政策與補助計畫 5) 未來挑戰與機會。最後產生一份結構化的 Word 文件（.docx），包含目錄、各章節標題、重點摘要和參考來源列表。
2️⃣ 按 Let's go →，觀察右側 Progress 面板的四個步驟：搜尋 → 整理 → 產生 Word → 驗證下載連結。
3️⃣ 完成後到 Working folder 點開產生的 .docx，體會「指令越具體（主題、來源數、章節結構、輸出格式），Claude 產出的報告品質越高」這個核心原則。

⚠️ Claude 搜尋的是網路上公開可取得的資訊，付費報告或內部資料需要你自己上傳到 Working folder 一起整合。
