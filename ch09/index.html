① 先回顧：三種設計武器

Canva Connector

「設計師的工具」
雲端平台整合

連接你的 Canva 帳號，自然語言描述需求，AI 呼叫 Canva API 生成多個候選方案，選定後存入帳號，可回 Canva 網站繼續手動編輯。

algorithmic-art Skill

「藝術家和數學家的玩具」
程式生成藝術

用 p5.js 產生互動式生成藝術。Cowork 先寫一份演算法哲學文件、推導能代表主題的數學概念，再實作成可互動的網頁作品。

canvas-design Skill

「博物館等級的工藝品質」
印刷品設計

用 Python 的 Pillow + reportlab，在 Cowork 的 VM 裡從空白畫布一筆一筆繪製。完全離線、不依賴外部 API，同時產出 PNG 與 PDF。

③ 接回真實操作：到 Cowork 做一次 demo

光做題不夠，打開 Claude Desktop → 切到 Cowork 分頁，把書中 algorithmic-art 範例的任務貼進去實際跑一次（不需要額外連接任何外部帳號）：

1️⃣ 在任務輸入區貼上：
請用 algorithmic-art Skill 幫我產生一個 p5.js 互動式生成藝術作品。主題是「數位花園」，用粒子系統模擬花朵生長的效果，配色使用粉色、綠色和金色。需要有互動功能，讓使用者可以用滑鼠點擊產生新的花朵。請產生一個完整的 HTML 檔案，可以直接在瀏覽器中開啟。
2️⃣ 按 Let's go →，觀察右側 Progress 面板跑完五個步驟：讀範本 → 建立演算法哲學 → 推導概念種子 → 實作 p5.js 互動藝術 → 儲存並驗證。
3️⃣ 完成後打開產出的 HTML 檔案，點畫布看花朵冒出、隨風搖曳，再回頭讀 Claude 寫的演算法哲學文件，體會「先思考再動手」——先定義用什麼數學概念表達主題，才開始寫程式碼。

⚠️ Cowork 目前是研究預覽版（research preview），若要改跑 Canva Connector 範例，記得先到 Settings > Customize > Connectors 連接 Canva 帳號。
