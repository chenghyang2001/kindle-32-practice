第 13 章 · 進階自訂與品牌設計

互動練習｜出自《Claude Cowork：31 個超好用範例實作》· 胡嘉璽著

① 先回顧：三個進階武器

skill-creator · 打造工具

「幫你建立新的 Skill」
從零開始的自訂 Skill 產生器

用自然語言描述需求，它會用 AskUserQuestion 追問細節，然後撰寫 SKILL.md、建立測試案例、執行測試、打包成 .skill 檔案。一次建立，無限重用。

brand-guidelines · 品牌一致

「自動套用品牌風格」
不用自己記色碼、字型

根據品牌風格指南，自動套用一致的配色、字型與設計元素。不太需要追問，也不用建立測試案例，因為指南本身已經定義好完整規範。

Canva Connector · AI 設計引擎

連接 Canva 的設計引擎
不是產生檔案，是呼叫 API

把需求傳到 Canva 的 generate-design-structured API，由 Canva 的 AI 設計引擎產生多個專業視覺方案，成品存在你的 Canva 帳號、可到網頁版繼續編輯。

一句話：skill-creator 打造工具、brand-guidelines 統一風格、Canva Connector 借力 AI 設計引擎。本章示範的正是 Cowork「不只是使用工具，而是打造工具」的終極應用。

③ 接回真實操作：到 Cowork 做一次 demo

光做題不夠，打開 Claude Desktop → 切到 Cowork 分頁，把書中最經典的自訂 Skill 範例貼進去，實際跑一次「用 skill-creator 建立台灣發票整理工具」：

1️⃣ 在任務輸入區貼上：
請使用 /skill-creator 幫我建立一個自訂 Skill，名稱叫做「台灣發票整理」。這個 Skill 的功能是：讀取使用者提供的電子發票截圖或文字資料，自動辨識發票日期、品項、金額，然後整理成一份結構化的 Excel 表格，包含日期、商家、品項、金額、分類（餐飲/交通/日用品等）欄位。請幫我完成 Skill 的建立、測試和部署。
2️⃣ 按 Let's go →，依序回答 skill-creator 追問的四個問題（輸入來源、費用分類、Excel 額外功能、是否設定測試案例），然後放著讓它自動跑完六個步驟——閱讀參考、撰寫 SKILL.md、建立測試案例、執行測試、根據回饋修改、打包成品。
3️⃣ 測試通過後點 Save skill，把它存進你的 Cowork Skills 清單。之後任何時候要整理發票，只要任務描述提到相關關鍵字，Claude 就會自動載入這個 Skill——這就是「一次建立，無限重用」。

⚠️ skill-creator 建立過程中會用到 Desktop Commander 連接器建立目錄與執行 Python 程式，第一次跑會請你確認權限，逐一放行即可。
